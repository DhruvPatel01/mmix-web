About This Project

This is a web-based toy debugger for stepping/running through MMIX code in the browser (using WebAssembly). I created this mostly for my own use, to learn MMIX instructions.

Workflow

You can write your MMIX code in the editor or load it from GitHub. If you load it from GitHub, it automatically gets compiled. You can press compile to recompile. After compiling, you can use `step` to step through the code one instruction at a time. There isn't much support for breakpoints. You
